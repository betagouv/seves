Modification d'une fiche clôturée

Vous êtes sur le point de modifier une fiche déjà clôturée. Cette action est possible pour les utilisateurs de l’administration centrale.
Toute modification peut avoir un impact sur les valorisations déjà réalisées (statistiques, tableaux de bord, exports).
Merci de vérifier que cette modification est nécessaire avant de poursuivre.
